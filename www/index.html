MkvDefaultTrackSwitcher

A simple open-source tool to automatically set the default audio and subtitle tracks in MKV video files.

Download for Windows Download for Linux (GTK) Download for MacOS
Star on Github

Features

Multiple files

Batch process multiple MKV files at once, provided they have the same order of subtitle / audio tracks

Easy to use

Simple UI, no bells and whistles

Fast

Great performance because it doesn't need to process the whole file.
